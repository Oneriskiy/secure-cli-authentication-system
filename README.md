# secure-cli-authentication-system
___
## Описание
### Проект реализует систему регистрации и аутентификации пользователей в CLI-приложении. Пароли пользователей хешируются при регистрации и проверяются по хэшу при входе в систему.

___

>все нужные зависимости находятся в `requirements.txt` и инструменты разработчика в `dev-requirements.txt`

## Установка
___
```

pip install -r requirements.txt
```
## Запуск
___
```
python main.py
```
